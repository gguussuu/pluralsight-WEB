# pluralsight-WEB
> tailwindcss를 활용하여 pluralsight.com 사이트의 첫페이지를 구현하는 프로젝트입니다.

## 사전과제
-  https://www.pluralsight.com/ 이 사이트의 첫페이지를 구현
- https://tailwindcss.com/ 사용하여 구현
- 애니메이션은 구현할 필요없음
- 이미지나 로고는 사이즈 맞는 다른 이미지로 대체 가능함
- 기한이 넘어가면 현재 구현된 부분까지만 제출
